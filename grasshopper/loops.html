Loops

Loops are used to run the same block of code multiple times. There are many different types of loops in JavaScript.

For...of Loop

A for...of loop runs the same block of code {} for each element in an array or string.

Classic for Loop

A classic for loop runs a block of code {} a specific number of times.

While loop

A while loop repeats the code inside of its block {} until the test in its parentheses is no longer true.

Nested Loop

A nested loop is a loop inside another loop. Although nested loops can be useful, too many can make code execution slow or difficult to understand.

For..in Loop

A for...in loop runs a block of code {} for each property in an object. For..in loops are similar to for...of loops, except they loop through objects.
