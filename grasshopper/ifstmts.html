If Statements

If Statements are a common tool for controlling code flow. They run code based on whether a condition, or test, is true.

If Statement

If statements allow you to control if a piece of code runs. If the test inside the parentheses () is true, then the code inside the block {} runs.

If...else Statement

If...Else statements allow you to run one block of code if a condition is true, or else run a different block of code. If the test is true, the if block {} runs. If the test is false, the else block {} runs.

Ternary Operator

The ternary operator uses a question mark ? and a colon : to replace an entire if...else statement with a single line of code. For example: test ? doThisIfTrue : elseDoThis.
