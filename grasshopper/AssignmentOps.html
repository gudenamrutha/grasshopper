Assignment Operators

Assignment operators are used to give values to variables. They set the value on the right of the assignment operator to the variable on the left.

Assignment Operator

The = operator assigns a value to a variable.

Addition Assignment Operator

The += operator adds the value on the right of the operator to the variable, and assigns the result to the variable.

Subtraction Assignment Operator

The -= operator subtracts the value on the right of the operator from the variable, and assigns the result to the variable.

Multiplication Assignment Operator

The *= operator multiplies the variable by the value on the right of the operator, and assigns the result to the variable.

Division Assignment Operator

The /= operator divides the variable by the value on the right of the operator, and assigns the result to the variable.

Remainder Assignment Operator

The %= operator divides the variable by the value on the right of the operator, and assigns the remainder to the variable.

Increment Operator

The ++ operator adds 1 to an existing numeric value. It’s shorthand for += 1.

Decrement Operator

The -- operator subtracts 1 from an existing numeric value. It’s shorthand for -= 1.
