Variables

Variables allow you to reference the same piece of information multiple times. In JavaScript, variables are defined using var, let, or const.

Var

The var keyword tells the computer to create a variable. The name of the variable needs to immediately follow the var keyword.

Let

A variable created with let has a local scope, meaning it can only be used inside the code block {} it was created.
