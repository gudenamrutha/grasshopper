Comparison Operators

Comparison operators are used to compare values. They return true or false based on whether a comparison is true or false.

Strict Equality Operator

The === operator returns true when 2 values (numbers, strings, variables, etc.) are strictly equal to each other. To be strictly equal, the values must be equal and have the same data type.

Strict Inequality Operator

The !== operator returns true when 2 values (numbers, strings, variables, etc.) are strictly not equal to each other. To be strictly not equal, the 2 values must be different, be different data types, or both.

Loose Equality Operator

The == operator returns true when 2 values (numbers, strings, variables, etc.) are loosely equal to each other. To be loosely equal, the values must be the same (data type doesn’t matter).

Loose Inequality Operator

The != operator returns true when 2 values (numbers, strings, variables, etc.) are loosely not equal to each other. To be loosely not equal, the values must be different (data type doesn’t matter).

Greater Than Operator

The > operator compares whether the left side is greater than the right side, and returns true if it is.

Less Than Operator

The < operator compares whether the left side is less than the right side, and returns true if it is.

Greater Than or Equal to Operator

The >= operator compares whether the left side is greater than or equal to the right side, and returns true if it is.

Less Than or Equal to Operator

The <= operator compares whether the left side is less than or equal to the right side, and returns true if it is.
